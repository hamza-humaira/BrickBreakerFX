# BrickBreakerFX

A JavaFX-based Brick Breaker game inspired by the classic arcade game.  
The player controls a paddle to bounce a ball and destroy bricks. The game features multiple colored bricks, score tracking, sound effects, and a stylish dashboard for game over or victory.

## Features

- Paddle movement with **keyboard (left/right)** and **mouse control**  
- Ball physics with realistic bounce  
- Multiple rows of colored bricks  
- Score tracking  
- Game Over and Victory screens with animations  
- Sound effects for brick hits, victory, and game over  
- Smooth dashboard with fade and scale transitions  
- Restart (**R**) and Quit (**Q**) functionality  

## Technologies Used

- Java 17+  
- JavaFX 25+  
- **AudioClip** for sound effects  
- Developed in **IntelliJ IDEA**  
- JDK from **BellSoft**

## How to Play

1. Move the paddle using **Left/Right arrow keys** or **mouse**.  
2. Break all the bricks to **win the game**.  
3. If the ball falls below the paddle, **game over**.  
4. Press **R** to restart the game or **X** to quit.
